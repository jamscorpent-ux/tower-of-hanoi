# tower-of-hanoi
A repository containing essential algorithmic implementations in Python. Includes sorting (Selection, Merge), searching (Linear, Binary), and the classic Tower of Hanoi puzzle. Designed for learning and understanding fundamental CS concepts.

# 🚀 Algorithmic Implementations

Welcome to my collection of essential Computer Science algorithms! This repository serves as a practical guide to mastering core programming logic, covering foundational sorting, searching, and puzzle-solving techniques.

## 🛠 Features
- **Searching:** Master efficient data retrieval with:
  - 🔍 **Linear Search**: Sequential scanning.
  - ⚡ **Binary Search**: Fast searching in sorted lists.
- **Sorting:** Learn algorithmic efficiency with:
  - 📉 **Selection Sort**: Simple comparison-based sorting.
  - 🧩 **Merge Sort**: Fast "Divide and Conquer" sorting.
- **Puzzles:** 🗼 **Tower of Hanoi** – An elegant recursive implementation.

## 💡 Why this repo?
Each implementation is built with a focus on **clean code** and **algorithmic complexity** ($O(n)$ notation). Perfect for interview prep, academic review, or just sharpening your logic!
